# Official Antigravity Color & Light Manifesto

## Theme: Emisson vs Painting
Colors are not "painted" on surfaces; they are emitted from them. Light represents energy; darkness represents depth.

## Core Variables
- **void**: #080808 (Deep Space background)
- **glass**: rgba(26, 26, 26, 0.6) (Glassmorphism base)
- **kinetic**: #00FF9C (Mint Action - Success, Active Lifting)
- **logic**: #7000FF (AI Purple - Intelligence, Calculations)
- **ghost**: rgba(237, 237, 237, 0.6) (Subdued Text)

## Surface Rules
1. **Base Vacuum**: Never use pure #000000. Use `#080808` with a 5% opacity radial gradient in the top-right corner.
2. **Weightless Borders**: Use 1px top-down gradient borders.
   - Top: `rgba(255, 255, 255, 0.15)`
   - Bottom: `rgba(255, 255, 255, 0.02)`
3. **Glassmorphism**: Always use `backdrop-filter: blur(16px)` on cards and modals.

## Kinetic Accents
- Any element in **Kinetic Green** must have a glow: `0 0 15px rgba(0, 255, 156, 0.4)`.
- Use **Logic Purple** for deep info and progress charts.
