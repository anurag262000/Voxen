# Antigravity Logic & Interaction Rules

## Transitions
- **Float-in Rule**: All components must use "Float-in" (Slide up + Fade in) animations.
- **SNAP-IN**: Never "Snap-in." All entries should feel weightless.

## Interaction Principles
- **Kinetic Green Hover**: When a primary action (like SignIn) is hovered, the background nebula should slightly shift toward the element.
- **Pulse Wave**: Use for background calculation states (e.g., when the Python engine is processing routine data).
- **Voice Sphere States**:
  - **Idle**: Dimmed Gray, 1px White border.
  - **Listening**: Kinetic Green outer ring glow.
  - **Processing**: Violet Logic spinning ring.

## Energy States
- **High Energy (Heavy Set)**: Vibrant Green borders.
- **Recovery (Low Energy)**: Soft/Dimmed Violet.
