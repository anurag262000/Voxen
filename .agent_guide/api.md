# Antigravity Engine & API Integration

## Engine UI (Floating Orb)
- When the Python engine is processing routine data, show a **Floating Orb** animation (Secondary Logic Purple - #7000FF).
- The Orb should "Pulse" to indicate energy consumption.

## Voice Logging Pipeline
- **Transcription**: Appear in center, 0.8 opacity, textMain (#EDEDED).
- **Fly-up**: Once confirmed/processed, transcription must "fly/float" upward into the log list.
- **Microphone**: Bottom center circle.

## Routine States
- If Python engine suggests a **Heavy** set: Vibrant Green borders.
- If Python engine suggests a **Recovery** set: Dimmed/Soft Violet.

## API Standards
- Always separate Logic from Display. Data should be "Weightless" until it hits the Kinetic Green action.
- Feedback to the User: Pulse waves across the screen when typing measurements or processing.
